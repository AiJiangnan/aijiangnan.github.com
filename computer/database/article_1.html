MySQL

MySQL是一个关系型数据库，MySQL所使用的SQL语言是用于访问数据库的最常用标准化语言。MySQL软件采用了双授权政策，它分为社区版和商业版，由于其体积小、速度快、总体拥有成本低，尤其是开放源码这一特点，一般中小型网站的开发都选择MySQL作为网站数据库。由于其社区版的性能卓越，搭配PHP,Linux和Apache可组成良好的开发环境，经过多年的web技术发展,在业内被广泛使用的一种web服务器解决方案之一,称之为LAMP。

MySQL安装与配置

安装版比较简单，根据安装向导即可完成，我们主要讲免安装版的配置。

  1. 解压下载好的ZIP安装包到D盘（随便什么盘），为了方便把文件夹命名为mysql，记住安装路径。
  2. 在mysql文件夹下找到一个.ini文件（名字一般是my_default.ini），把这个文件复制到C:\windows\文件夹下，并改名为my.ini。
  3. 修改my.ini文件，用记事本打开，在“[mysqld]”这个组中加入如下两条。

    basedir = "D:/mysql/"

    datadir = "D:/mysql/data/"

    basedir参数表示MySQL软件的安装路径，datadir参数表示MySQL数据库中数据文件的存储位置。除了上述内容外，在文件后面再加入一个组和一条记录，内容如下：

    [WindowsMySQLServer]

    Server = "D:/mysql/bin/mysqld.exe"

  4. 配置系统环境变量，右击我的电脑->属性->高级系统设置->环境变量中的Path添加mysql文件夹下的bin文件夹的路径。
  5. 设置MySQL服务，在命令提示符中（可能要管理员权限）输入mysqld -install，显示安装成功即可。
  6. 启动和关闭服务。

    net start mysql

    net stop mysql

    或者在电脑运行中输入services.msc打开电脑服务，打到MySQL服务并启动。
