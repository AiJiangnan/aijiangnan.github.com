运算符与表达式

算术运算符

  • + ：加法运算符或正值运算符，如 3+4，23+78，+13等。
  • - ：减法运算符或负值运算符，如 11-2，-5等。
  • * ：乘法运算符，如 58*45，-2*12等。
  • / ：除法运算符，如 123/45等。
  • % ：模运算符，或称为求余运算符，只能用于整型数据，如 17%4，44%3等。

自增自减运算

自增运算符（++）与自减运算符（--）只能用于变量，不能用于常量或表达式，它们的作用是使运算对象加1可减1。如 i++ 等价于 i=i+1 ，同理 i-- 等价于 i=i-1。

  • ++i ：i 自增 1 后再参与其他运算。
  • --i ：i 自减 1 后再参与其他运算。
  • i++ ：i 参与运算后，i 的值再自增 1 。
  • i-- ：i 参与运算后，i 的值再自减 1 。

分析以下程序的输出结果：

#include"stdio.h"
void main(){
  	int a = 1, b = 2;
  	printf("%d,%d\n",++a,b++);
  	printf("%d,%d\n",a--,--b);
}

分析：第一个输出语句需要输出 ++a 和 b++ ，所以我们可知 ++a 的值为2而 b++ 的值也为2，因此这个输出语句将会输出2，2。第二个输出语句需要输出 a-- 和 --b ，但需注意的是此时的 a 和 b 由于之前的 ++a 和 b++ 已经分别变成了2和3，于是 a-- 的值为2而 --b 的值也为2，因此将会输出2，2。综上，这个程序最后会输出：

2,2
2,2

逗号运算符

在C语言中逗号“ , ”也是一个运算符，称为逗号运算符。其功能是把若干个表达式连接起来组成一个表达式，称为逗号表达式。其一般形式为：

表达式1, 表达式2, …… ,表达式 n

逗号运算符的优先级是所有运算符中最低的。

#include"stdio.h"
void main(){
		int a = 2,b = 4,c = 6,x,y;
		y = ((x=a+b),(b+c));
		printf("y=%d,x=%d\n",y,x);
}

运行结果输出：

y=10,x=6

赋值运算符

赋值运算符级别较低，结合方向为自右向左。

  1. 简单赋值运算符

    2. 简单赋值运算符为“ = ”。由“ = ”连接的式子称为赋值表达式。其一般形式：

    变量 = 表达式

    例如：

    y = a + b
w = sin(a)+sin(b)
x = i++

    赋值运算符的功能是计算表达式的值给左边的变量。赋值运算符具有右结合性。因此a=b=c=5可理解为
    a=(b=(c=5))。

  2. 复合赋值运算符

    3. 如a+=b等于a=a+b，x/=a+b等于x=x/(a+b)，使用复合赋值运算符，可以简化代码，还可以提高编译效率。

    在C语言中，赋值“ = ”和等于“ == ”是两个不同的的运算符。

条件运算符

这是一个三目运算符，用于条件求值。其一般形式：

表达式1 ? 表达式2 : 表达式3

运算规则如下：先计算表达式1的值，如果值为真（非0），则取表达式2的值为整个条件表达式的值，否则表达式3的值为整个条件表达式的值。

例如：z=a>b?a:b

因为条件运算符的优先级别比赋值高，所以该表达式表示：如果a大于b，则取a的值赋给z，否则取b的值赋给z。
